
SuperToasts Library
=================

The SuperToasts library is a customizable way for a developer to interact with his/her users. 
This library seeks to give developers a better way of communicating with their users.

![Example Image][3]

Try out the sample applications on the Android Market: [Feature Demos][4],
[Fragments][5], and [RoboGuice][6].

Continuous integration is provided by [Travis CI][7].
